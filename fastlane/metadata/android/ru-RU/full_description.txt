openHAB - "empowering the smart home" - vendor and technology agnostic open source home automation

Версия для F-Droid поддерживает просмотр карт. GCM и отчёты о сбоях удалены. Приложение не будет получать уведомления от openHub Cloud.

<b>Важное примечание</b>

Для этого приложения необходим сервер openHAB

openHAB - это основанная на Java платформа домашней автоматизации с открытым исходным кодом, интегрирующая и комбинирующая в себе большое количество различных систем "умного дома" в единое решение.
On top of the unified abstraction layer all connected devices are available for the overarching automation rule engines and different user interfaces.

<b>Поддерживаемые продукты</b>

 Для примера: Z-Wave, Philips Hue, Amazon Echo, Chromecast и Sonos.
